The 'V5' visual overhaul replaces the purple/green Yoast-inspired palette with a Meta/Facebook-aligned design system. Primary color is Facebook blue (214 89% 52%), accent is soft indigo/violet (253 63% 57%). Backgrounds are clean white with soft gray (220 14% 96%) for alternating sections. Typography is significantly larger: h1 up to 7xl, h2 up to 2.75rem, body text base/lg. Hero sections use blue-to-indigo gradients instead of purple-to-green. Cards use subtle shadows, rounded-2xl corners, and minimal borders. The dashboard mockup in the hero uses a browser chrome frame with score rings and metric bars. All marketing pages follow the same Meta-inspired language: trustworthy, polished, platform-grade.
